## Steps to run this locally:
1.) ``open a terminal/command prompt and type: git clone https://github.com/0Armaan025/xquotes``
2.) ``cd xquotes``
3.) ``npm install``
4.) ``npm run dev`` and boom! it will be running on ``http://localhost:3000`` :)
